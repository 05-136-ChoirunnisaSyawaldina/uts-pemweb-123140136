# Book Library - UTS Pengembangan Aplikasi Web

Aplikasi web "Book Library" ini adalah submission untuk Ujian Tengah Semester (UTS) mata kuliah Pengembangan Aplikasi Web RA. Aplikasi ini dibuat menggunakan ReactJS dan terhubung dengan Open Library API. Fitur utamanya memungkinkan pengguna untuk mencari buku (berdasarkan judul/penulis), memfilter hasil (berdasarkan subjek, tahun, dll.), melihat detail buku (deskripsi & subjek), serta mengelola "Reading List" pribadi. Fitur reading list ini dapat menambah dan menghapus buku, dan datanya disimpan di localStorage agar tidak hilang saat sesi ditutup.
